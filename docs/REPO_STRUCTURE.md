# Repository Structure

## Root

- `README.md`: public landing page and master category table.
- `docs/`: generated navigation and learning workflow.
- `CONTRIBUTING.md`: contribution rules.
- `CODE_OF_CONDUCT.md`: collaboration expectations.
- `LICENSE`: repository license.
- `.github/`: issue and pull request templates.
- `.opencode/`: local generation scripts and tooling metadata.

## Numbered Categories

Numbered folders are the primary interview syllabus.

```text
001. JavaScript Core/
002. JavaScript Internals/
...
110. Staff-Principal Engineer Mindset/
```

## Senior Add-On Categories

Unnumbered folders hold production and senior-level expansion topics.

```text
Production Readiness/
Operational Excellence/
Resilience Patterns/
...
Architecture Governance/
```
