# Contributing

This repository is a structured engineering knowledge base. Contributions should improve clarity, depth, correctness, examples, or production relevance.

## Contribution Types

- Add missing edge cases.
- Add code examples.
- Improve production scenarios.
- Add debugging strategies.
- Add architecture trade-offs.
- Fix broken links or formatting.

## Content Standards

- Prefer practical explanations over generic definitions.
- Include why, how, and when.
- Include failures, scaling limits, and debugging signals.
- Keep examples small and executable where possible.
- Avoid vendor-specific advice unless the topic requires it.

## Pull Request Checklist

- The changed topic still follows the repository file structure.
- Markdown links work.
- Code examples are readable and scoped.
- Production claims include failure modes or trade-offs.
- Generated files were regenerated when structure changed.
